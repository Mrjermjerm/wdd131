Website Plan

BYUI Newsletter: This is a website for BYUI newsletter meant to be appealing and informative.

Site Name

The site name is BYUI Newsletter. This name was chosen because it directly reflects the focus of the website, which is about BYU-Idaho's events and news.

Optional Domain: byuinewsletter.com

Site Purpose

The purpose of the site is to inform students and teachers of events that are taking place on and around campus, in a way that is easy and informative.

Scenarios

  • What if I want to stay up to date with groups around me?
  • What makes this website useful compared to what BYU-Idaho currently has?

Color Schema

Primary: BYUI Blue (#0d74b9) Crimson Red: #9B1B30 White: #FFFFFF

Secondary: Light Gray: #D1D1D1 Charcoal Gray: #333333

Accent: Golden Yellow: #FFCC00

Typography

The site will use the following fonts:

  • Arial, sans-serif: Used for body text and general content.
  • Roboto, sans-serif: Used for headings and important UI elements.

Wireframe

The wireframe for the home page includes the following elements:

  • Header with logo and navigation bar
  • Main content section related to current page
  • Footer with links to social media and author name with date
